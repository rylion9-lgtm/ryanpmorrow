# Hello! I'm Ryan Morrow

Welcome to my data science portfolio. I’m a Statistics: Data Science student at Brigham Young University interested in using data to understand complex problems and make better decisions.

This site highlights projects where I apply programming, statistics, and visualization to real datasets.

## About This Portfolio

Each project on this site shows a different part of the data science process:

- Collecting and preparing data
- Exploring trends and patterns
- Creating visualizations
- Communicating insights clearly

I built this site using [Quarto](https://quarto.org/) and host it on [GitHub Pages](https://pages.github.com/).

## Skills I’m Building

- **Programming**: Python, Pandas, NumPy
- **Data Visualization**: Matplotlib, Seaborn
- **Statistical Analysis**: Regression, hypothesis testing, exploratory analysis
- **Tools**: Jupyter Notebooks, Git, GitHub, Quarto
